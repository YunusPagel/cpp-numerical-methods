# C++ Numerical Methods

A collection of small C++ programs implementing basic numerical algorithms.

## Included Programs
- **Newton Method** – Root finding for functions.
- **Bisection Method** – Stable root-finding approach.
- **Cosine Similarity** – Calculates similarity between two vectors.

## Purpose
Practical exercises in C++ programming and numerical algorithm implementation.

## Technologies
- C++
- Standard Template Library (STL)
- Command-line interface programs
